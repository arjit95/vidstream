#### Vidstream [WIP]
A full featured streaming service on kubernetes

#### Running
This guide assumes you have a running kubernetes cluster and have > 8GB RAM.

- Install [skaffold](https://skaffold.dev)
- Run `requirements.sh`
- Run `skaffold dev`
